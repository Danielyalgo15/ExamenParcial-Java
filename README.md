# ExamenParcial-Java

Repositorio del examen parcial desarrollado en Java utilizando recursión y la técnica divide y vencerás.

## Estructura del proyecto

- Problema 1:
  - Verificar si un arreglo está ordenado utilizando recursión.

- Problema 2:
  - Encontrar el segundo elemento más grande utilizando divide y vencerás.

- Problema 3:
  - Contar inversiones en un arreglo utilizando merge sort.
